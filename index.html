Graffix is an idea I had which aimed to merge my interest in photographing some of DC's graffiti during my walks around the city, and a desire to learn how to impliment Google Maps into one of my projects. It's built on a backend server developed with the Ruby on Rails framework and a React front end. See the deployed version here.

Imagini is an attempt to develop a prototype that mimicked Instagram, but with a focus on users involved with photography. It's built with an Epress API with server-side view templates made with Handlebars. See the deployed version here.

Dollar Beer Club was a group project at General Assembly, made by myself, Sandro Guraspasvili, Alonzo Moses, and Marquet Reid. It's a site where users can buy beers. The selection is generated by connecting to a third party API. It's built with Mongoose, Express, React and Node. See the back end and front end repos for the source code. See a deployed version here.
